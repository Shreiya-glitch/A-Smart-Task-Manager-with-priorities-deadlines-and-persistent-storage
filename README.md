# Smart Task Manager (Python CLI)

A command-line task manager built with Python — designed to help you stay on top of your work with deadlines, priorities, and a few smart tricks up its sleeve.

---

## The Problem

We’ve all been there — juggling too many tasks, forgetting deadlines, and losing track of what actually matters. This project is a simple fix for that. It gives you a structured, no-fuss way to manage your day without opening a browser or paying for a subscription.

---

## What It Can Do

- Add, edit, and delete tasks with ease
- Set deadlines so nothing slips through the cracks
-  Assign priority levels — High, Medium, or Low
-  Get a quick productivity snapshot (completed vs. still pending)
-  Automatically flags tasks that are overdue
-  Surfaces tasks due within the next 3 days so you’re never caught off guard
-  Saves everything to a JSON file — your tasks are still there when you come back
-  Sorts tasks smartly — pending ones first, ordered by priority

---

## Built With

- *Python* — the backbone of the whole thing
- *JSON* — lightweight, human-readable storage (no database needed)
- *Datetime* module — powers all the deadline and overdue logic
